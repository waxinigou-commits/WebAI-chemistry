# WebAI-chemistry

一个前端优先的化学学习网站原型仓库，用来逐步探索“交互式化学实验网页”这个方向。

## 预览地址

- GitHub Pages: https://waxinigou-commits.github.io/WebAI-chemistry/

## 技术栈

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages

## 版本说明

### v0.6.0 · 实验器材画布版

本版本继续彻底收敛方向，不再做“实验主题页面”，而是直接改成“实验器材画布版”。目标是更接近 NB 实验室那种体验：中心是实验画布，右侧是器材与药品库，底部是步骤与状态，用户第一眼看到的是器材、装置和实验空间，而不是说明区。

#### 本版本新增与调整

- 页面主形态从“实验平台页面”重构为“实验器材画布”
- 中央区域升级为近似全屏的实验画布，突出：
  - 铁架台
  - 主反应玻璃管
  - 酒精灯
  - 尾气处理试管
  - 药品瓶与实验小工具
- 页面右侧改为器材与药品库，分为：
  - 仪器
  - 药品
  - 工具
- 页面底部改为实验步骤条 + 状态栏 + 观察记录
- 顶部仅保留轻量工具栏，不再用大标题和大说明抢占实验空间
- 当前实验主题切换为更接近 NB 风格装置表达的：
  - 一氧化碳还原氧化铜
- 页面交互逻辑继续弱化说明卡片，强化“选择器材 / 执行步骤 / 观察装置变化”
- 通过步骤切换让画布呈现：
  - 搭装置
  - 点燃酒精灯
  - 观察实验现象

#### 本版本目的

- 验证“实验器材画布”是否比“实验平台页面”更接近目标参考产品形态
- 让用户第一视觉关注点回到实验装置本身
- 为后续增加拖拽、摆放、连接、点火、局部观察等交互打底

#### 下一步建议

- 增加真正的拖拽摆放，而不是仅做状态切换
- 增加器材连接关系与装置组装逻辑
- 增加局部放大观察、实验镜头和微观/宏观切换
- 增加更真实的动画过程，如导气、受热、颜色变化、尾气处理变化
- 后续再扩展更多化学实验装置和器材类别

### v0.5.0 · 化学平衡实验平台版

本版本不再把产品定位成“交互学习页面”，而是明确朝“实验平台”方向重构。页面中心从信息面板切换为实验工作台，强调试剂、容器、环境控制和实验记录，让用户感觉自己在操作一个实验平台，而不是只在看解释。

#### 本版本新增与调整

- 页面整体从“三栏学习台”重构为“实验平台工作台”结构
- 中央区域升级为实验台主舞台，加入：
  - 左侧实验架
  - 双容器反应区
  - 中央平衡状态面板
  - 右侧环境控制台
  - 底部实验台面
- 首版实验动作改为对象化操作：
  - 加入反应物
  - 升高温度
  - 增大压强
- 左侧从“变量控制”改为“试剂与实验材料”面板
- 右侧从“理解 tab”改为“实验记录与状态”面板
- 页面中的主操作逻辑从“改参数看解释”改为“执行实验动作看系统变化”
- 新增实验架、控制台、设备工具栏、实验步骤与观察记录等平台化元素
- 保留化学平衡主题，但产品表达明显向 NB / 实验平台方向靠拢

#### 本版本目的

- 验证“像实验平台”而不是“像交互学习页面”的方向是否更成立
- 建立实验对象、实验动作、实验环境三者之间的统一表达
- 为后续继续加入更真实的操作路径、器材对象和实验流程打基础

#### 下一步建议

- 增加更真实的器材对象与点击/拖拽操作
- 增加加样、升温、增压的动画过程与时间维度
- 增加实验失败路径、步骤错误提示与实验回放
- 增加仪表读数、颜色变化、气泡等更真实的实验反馈
- 后续继续扩展到更多化学实验主题与实验台场景

### v0.4.0 · 化学平衡交互学习台

本版本从“数字实验室场景”继续往前推进，开始围绕“化学平衡”做真正的交互式学习页面，不再只是演示实验，而是让用户先预测、再调变量、再理解平衡为什么移动。

#### 本版本新增与调整

- 页面主题从“二氧化碳制取与检验”切换为“化学平衡”学习单元
- 整体结构升级为三栏学习实验台：
  - 左侧条件调节区
  - 中间平衡变化主舞台
  - 右侧理解面板
- 新增 3 个首版核心变量主题：
  - 浓度
  - 温度
  - 压强
- 新增预测题机制：用户先做判断，再提交预测，再应用变化验证
- 新增宏观现象、微观变化、原理解释三类同步内容
- 新增平衡移动状态、正逆反应速率、方向提示等核心反馈
- 底部新增学习记录区，用于沉淀宏观 / 微观 / 原理三层理解
- 页面表达从“做实验演示”升级为“理解化学变化机制”

#### 本版本目的

- 验证“交互式学习实验台”是否比传统实验演示页更有产品价值
- 把化学学习从“看现象”推进到“理解为什么发生”
- 为后续继续做单元化、课程化交互学习页面打基础

#### 下一步建议

- 把当前原型继续细化成更真实的化学平衡过程动画
- 增加错误预测后的针对性纠错路径
- 增加单元切换后的学习进度保存
- 增加题目结果统计与教师视角
- 后续可继续扩展到反应速率、酸碱滴定等主题

### v0.3.0 · 数字实验室场景版

本版本不再把页面当成普通的内容页，而是把它重构成一个更像真实实验室的交互场景页面。

- 整体页面结构升级为“数字实验室”布局
- 新增顶部黑板区，用于展示：
  - 实验目标
  - 当前实验指引
  - 化学方程式
  - 自动演示 / 重置操作
- 中间主区域重构为实验室舞台：
  - 背景墙面
  - 黑板屏
  - 实验架
  - 实验桌面
  - 锥形瓶、导气管、石灰水容器
- 左侧新增原料区
- 右侧新增器材区
- 底部新增实验记录与引导区
- 继续保留 CO₂ 制取与检验的 4 步实验逻辑
- 用更场景化的方式呈现实验现象与操作路径

#### 本版本目的

- 确立“不是文章页，而是实验室页面”的产品方向
- 验证场景式布局是否更适合化学实验网页
- 为下一步加入更真实的实验操作交互打基础

#### 下一步建议

- 增加点击原料后触发更真实的操作反馈
- 增加轻量拖拽交互
- 增加器材高亮与错误路径提示
- 让黑板内容和实验桌变化联动更强

  - 实验目标
  - 器材与试剂
  - 当前实验现象
  - 反应原理
  - 注意事项
  - 产品化扩展方向

#### 本版本目的

- 验证“交互式化学实验页”是否适合作为产品核心方向
- 用较轻量的前端方式建立可扩展页面结构

## 版本管理约定

后续开发默认遵循以下规则：

- 每个可用阶段保留独立 git commit
- 关键阶段打 git tag，便于切换和回退
- 每实现一个新版本，同步更新本 README
- README 中持续记录当前版本能力、变更点和后续方向
