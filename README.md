# WebAI-chemistry

一个前端优先的化学学习网站原型仓库，用来逐步探索“交互式化学实验网页”这个方向。

## 预览地址

- GitHub Pages: https://waxinigou-commits.github.io/WebAI-chemistry/

## 技术栈

- HTML
- CSS
- Vanilla JavaScript
- GitHub Pages

## 版本说明

### v0.3.0 · 数字实验室场景版

本版本不再把页面当成普通的内容页，而是把它重构成一个更像真实实验室的交互场景页面。

#### 本版本新增与调整

- 整体页面结构升级为“数字实验室”布局
- 新增顶部黑板区，用于展示：
  - 实验目标
  - 当前实验指引
  - 化学方程式
  - 自动演示 / 重置操作
- 中间主区域重构为实验室舞台：
  - 背景墙面
  - 黑板屏
  - 实验架
  - 实验桌面
  - 锥形瓶、导气管、石灰水容器
- 左侧新增原料区
- 右侧新增器材区
- 底部新增实验记录与引导区
- 继续保留 CO₂ 制取与检验的 4 步实验逻辑
- 用更场景化的方式呈现实验现象与操作路径

#### 本版本目的

- 确立“不是文章页，而是实验室页面”的产品方向
- 验证场景式布局是否更适合化学实验网页
- 为下一步加入更真实的实验操作交互打基础

#### 下一步建议

- 增加点击原料后触发更真实的操作反馈
- 增加轻量拖拽交互
- 增加器材高亮与错误路径提示
- 让黑板内容和实验桌变化联动更强
- 后续再继续打磨 SVG 器材视觉精度

### v0.2.0 · 二氧化碳制取与检验交互原型

本版本把最初的 Hello World 首页升级成一个更接近真实产品方向的实验页原型，围绕“二氧化碳制取与检验”搭建了一个完整的交互式实验展示页面。

#### 本版本新增内容

- 以经典实验“二氧化碳制取与检验”为核心主题
- 提供可点击切换的 4 步实验流程
- 支持自动演示实验步骤
- 用前端动画模拟：
  - 锥形瓶中试剂反应
  - 气泡产生与通气过程
  - 澄清石灰水由清变浑浊
- 页面中整合了：
  - 实验目标
  - 器材与试剂
  - 当前实验现象
  - 反应原理
  - 注意事项
  - 产品化扩展方向

#### 本版本目的

- 验证“交互式化学实验页”是否适合作为产品核心方向
- 用较轻量的前端方式建立可扩展页面结构
- 先做产品表达与交互框架，不急着上后端

## 版本管理约定

后续开发默认遵循以下规则：

- 每个可用阶段保留独立 git commit
- 关键阶段打 git tag，便于切换和回退
- 每实现一个新版本，同步更新本 README
- README 中持续记录当前版本能力、变更点和后续方向
